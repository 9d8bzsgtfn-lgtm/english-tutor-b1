Grammar: Conditionals (Zero, First & Second)

IMPRESCINDIBLE - MEMORIZAR

Los tres condicionales principales

Conditional Uso Estructura
Zero (0) Verdades generales, hechos, leyes naturales If + present, present
First (1) Situaciones reales/probables en el futuro If + present, will + infinitive
Second (2) Situaciones hipoteticas o improbables If + past simple, would + infinitive

Zero Conditional: Verdades generales

ZERO CONDITIONAL

Estructura: If + present simple, present simple

If clause Main clause Significado
If you heat water to 100C, it boils. Ley cientifica
If it rains, the ground gets wet. Verdad general
If you don't water plants, they die. Hecho inevitable
If I eat too much, I feel sick. Habito/resultado habitual

Nota: Puedes usar WHEN en lugar de IF sin cambiar el significado:
When you heat water, it boils. = If you heat water, it boils.

First Conditional: Situaciones reales futuras

FIRST CONDITIONAL

Estructura: If + present simple, will + infinitive

If clause Main clause Significado
If it rains tomorrow, I will stay at home. Posibilidad real
If you study hard, you will pass the exam. Resultado probable
If she calls me, I'll tell her the news. Plan futuro
If we don't hurry, we won't catch the train. Advertencia

Variaciones con modales:

  • If you finish early, you can go home. (permiso)
  • If it snows, we might cancel the trip. (posibilidad)
  • If you're tired, you should rest. (consejo)

Second Conditional: Situaciones hipoteticas

SECOND CONDITIONAL - MUY IMPORTANTE

Estructura: If + past simple, would + infinitive

If clause Main clause Significado
If I had more money, I would travel the world. Situacion irreal presente
If I were you, I would accept the job. Consejo (hipotetico)
If she knew the answer, she would tell us. Situacion contraria a la realidad
If we lived near the beach, we would go swimming every day. Sueno/deseo

IF I WERE (no IF I WAS):
En el segundo condicional, usamos "were" para todos los sujetos (forma subjuntiva):
If I were rich... / If she were here... / If it were possible...

Unless = If... not

UNLESS

UNLESS significa "a menos que" = "if... not"

Con IF NOT Con UNLESS
If you don't hurry, you'll be late. Unless you hurry, you'll be late.
If it doesn't rain, we'll have a picnic. Unless it rains, we'll have a picnic.
I won't go if you don't come with me. I won't go unless you come with me.

Importante: UNLESS ya es negativo, asi que NO uses doble negacion:
Incorrecto: Unless you don't hurry...
Correcto: Unless you hurry...

Otros conectores condicionales

CONECTORES
Conector Significado Ejemplo
provided (that) siempre que, con tal de que I'll help you provided that you help me too.
as long as siempre que, mientras You can borrow my car as long as you're careful.
in case por si (acaso) Take an umbrella in case it rains.
even if aunque, incluso si I'll go even if it rains.

Orden de las clausulas

ORDEN Y PUNTUACION
Orden Ejemplo Puntuacion
IF + clausula primero If it rains, I'll stay home. COMA despues de la clausula IF
Clausula principal primero I'll stay home if it rains. SIN coma

Errores comunes a evitar

ERRORES FRECUENTES
Incorrecto Correcto Por que
If it will rain, I'll stay home. If it rains, I'll stay home. No uses WILL en la clausula IF (1st conditional)
If I would have money... If I had money... No uses WOULD en la clausula IF (2nd conditional)
If I was you... If I were you... Usa WERE para todos los sujetos (subjuntivo)
Unless you don't study... Unless you study... UNLESS ya es negativo, no anadir NOT
If I will be rich... If I am rich... / If I were rich... Presente (1st) o pasado (2nd), nunca WILL

Comparacion: First vs Second Conditional

DIFERENCIA CLAVE
First Conditional Second Conditional
If I have time, I will call you. If I had time, I would call you.
= Probablemente tendre tiempo = No tengo tiempo (situacion irreal)
If she asks me, I will help her. If she asked me, I would help her.
= Es probable que me pregunte = No creo que me pregunte

Pregunta clave: Es la situacion real/probable o hipotetica/improbable?

RESUMEN "SUBE LAS ANTENAS"
  • Zero: If + present, present (verdades generales)
  • First: If + present, will (situaciones reales futuras)
  • Second: If + past, would (situaciones hipoteticas)
  • NUNCA uses WILL/WOULD en la clausula IF
  • If I were (no "was") para consejos hipoteticos
  • Unless = If not (sin doble negacion)
