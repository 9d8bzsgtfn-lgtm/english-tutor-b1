Practice: Conditionals (Zero, First & Second)

INSTRUCCIONES

Selecciona la opcion correcta para completar cada oracion condicional.

Exercise 1: Zero Conditional

1. If you ________ ice in the sun, it ________.

leave / melts
will leave / melts
leave / will melt

Correct! Zero conditional uses present simple in both clauses for general truths.

2. When water ________ 0 degrees Celsius, it ________.

will reach / freezes
reaches / freezes
reached / would freeze

Correct! Scientific facts use zero conditional: present + present.

Exercise 2: First Conditional

3. If it ________ tomorrow, we ________ to the beach.

will rain / won't go
rains / won't go
rained / wouldn't go

Correct! First conditional: If + present simple, will + infinitive. Never use "will" in the if-clause!

4. You ________ the exam if you ________ hard enough.

pass / will study
passed / studied
will pass / study

Correct! First conditional for real future possibilities: will + infinitive / if + present.

5. If she ________ me, I ________ her the truth.

asks / will tell
will ask / tell
asked / would tell

Correct! First conditional: present in the if-clause, will in the main clause.

Exercise 3: Second Conditional

6. If I ________ rich, I ________ a yacht.

am / will buy
were / would buy
would be / bought

Correct! Second conditional: If + past simple, would + infinitive. Use "were" for all subjects.

7. If I ________ you, I ________ that job offer.

am / will accept
was / would accept
were / would accept

Correct! "If I were you" (not "was") is the standard form for giving advice.

8. What ________ you do if you ________ a million euros?

will / win
would / won
would / would win

Correct! Second conditional for hypothetical situations: would + infinitive / if + past simple.

Exercise 4: Unless and Mixed

9. ________ you hurry, you'll miss the train.

If
Unless
When

Correct! Unless = If not. "Unless you hurry" = "If you don't hurry".

10. I won't go to the party ________ you come with me.

if
unless
when

Correct! Unless = If not. The sentence means "I won't go if you don't come."
