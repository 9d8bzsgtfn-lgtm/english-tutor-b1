Speaking Practice: Conditionals

INSTRUCCIONES

Practica los condicionales Zero, First y Second para hablar de hechos, posibilidades y situaciones hipotéticas.

Objetivo: Usar el condicional correcto según la probabilidad y la realidad de la situación.

Section 1: Zero Conditional (Facts & General Truths)

EXERCISE 1

State scientific facts using zero conditional.

If + present, present (always true)

Model answer: "If you heat water to 100°C, it boils. If you mix blue and yellow, you get green. If plants don't get water, they die. If you freeze water, it becomes ice."

EXERCISE 2

Talk about general truths about yourself.

If I..., I always/usually/never...

Model answer: "If I drink coffee late, I can't sleep. If I don't eat breakfast, I feel tired. If I exercise in the morning, I have more energy. If I'm stressed, I eat chocolate."

EXERCISE 3

Give instructions using zero conditional.

If you want to..., you...

Model answer: "If you want to improve your English, you practice every day. If you want to save money, you spend less. If you want to be healthy, you eat well and exercise."

Section 2: First Conditional (Real Possibilities)

EXERCISE 4

Talk about your plans and their conditions.

If + present, will + verb (real future possibility)

Model answer: "If the weather is nice tomorrow, I'll go for a walk. If I finish work early, I'll call my friend. If I pass the exam, I'll celebrate. If it rains, I'll stay home."

EXERCISE 5

Make promises and warnings.

If you..., I'll... / If you don't..., you'll...

Model answer: "If you help me, I'll help you too. If you don't study, you won't pass. If you're late again, you'll be in trouble. If you tell me the truth, I won't be angry."

EXERCISE 6

Discuss real possibilities about the future.

Think about: career, studies, relationships, travel.

Model answer: "If I get a promotion, I'll buy a new car. If I save enough money, I'll travel to Japan. If my English improves, I'll look for international jobs. If I don't find a job here, I'll move to another city."

Section 3: Unless (= If not)

EXERCISE 7

Practice using "unless" instead of "if...not".

Unless = if not

Model answer:

  • "I won't go unless you come with me." (= if you don't come)
  • "Unless it stops raining, we'll stay home."
  • "You won't pass unless you study more."
  • "Unless I hear from you, I'll assume everything is OK."
EXERCISE 8

Convert these "if not" sentences to "unless":

  1. If you don't hurry, you'll be late.
  2. If it doesn't rain, we'll have a picnic.
  3. If I don't get a response, I'll call again.

Section 4: Second Conditional (Unreal/Hypothetical)

EXERCISE 9

What would you do if you won the lottery?

If + past, would + verb (imaginary situation)

Model answer: "If I won the lottery, I would quit my job. I would buy a big house. I would travel around the world. I would give some money to charity. I wouldn't waste it on silly things."

EXERCISE 10

If you could have any superpower, what would you choose?

Model answer: "If I could have any superpower, I would choose to fly. If I could fly, I would travel everywhere. I wouldn't need to take planes. It would be amazing to see the world from above."

EXERCISE 11

Talk about things you would change in your life if you could.

Model answer: "If I could change something in my life, I would learn more languages. If I had more time, I would exercise more. If I lived near the beach, I would swim every day. If I weren't so busy, I would read more."

EXERCISE 12

Use "If I were you..." to give advice.

Model answer: "If I were you, I would talk to your boss. If I were you, I wouldn't worry so much. If I were in your position, I would take the job. If I were you, I would apologize."

Section 5: Contrasting First vs Second Conditional

EXERCISE 13

Compare real vs unreal situations.

First = probable, Second = improbable/imaginary

Compare:

  • "If it rains tomorrow, I'll take an umbrella." (probable)
  • "If it rained in the Sahara, I would be surprised." (improbable)
  • "If I see John, I'll tell him." (possible)
  • "If I saw a ghost, I would scream." (hypothetical)
EXERCISE 14

For each topic, make one first and one second conditional sentence:

  • Getting a job
  • Winning money
  • Meeting someone famous

Section 6: Dreams and Wishes

EXERCISE 15

Talk about your dreams using second conditional.

Model answer: "If I could live anywhere in the world, I would live in New Zealand. If I could meet anyone from history, I would meet Leonardo da Vinci. If I could travel in time, I would visit ancient Rome."

EXERCISE 16

Complete these hypothetical sentences:

  • If I had a million dollars...
  • If I could speak every language...
  • If I were the president...
  • If I didn't have to work...

Section 7: Extended Monologue (2 minutes)

EXERCISE 17 - MONOLOGUE

Topic: "My ideal life"

Speak for 2 minutes about what your life would be like if it were perfect. Use second conditional:

  • Where you would live
  • What job you would have
  • How you would spend your time
  • Who you would be with
EXERCISE 18 - MONOLOGUE

Topic: "Plans for next year (if everything goes well)"

Use first conditional to discuss realistic possibilities.

Structure: "If everything goes well next year, I'll... If I manage to save money, I'll... If I pass my exams, I'll... Unless something goes wrong, I'll..."

EXERCISE 19 - MONOLOGUE

Topic: "If I could go back in time..."

What would you do differently?

Section 8: Role-Play Scenarios

EXERCISE 20 - ROLE-PLAY

Scenario: Giving advice to a friend

Your friend is stressed about a decision. Use "If I were you..."

EXERCISE 21 - ROLE-PLAY

Scenario: Planning a holiday

Discuss options using first conditional:

  • "If we go to the beach..."
  • "If we choose the mountains..."
  • "If the weather is bad..."
EXERCISE 22 - ROLE-PLAY

Scenario: Job interview

Answer: "What would you do if you got this job?"

Section 9: Quick Response Drills

QUICK DRILL

Complete quickly with the correct conditional:

  1. If water reaches 100°C, it... (fact)
  2. If it rains tomorrow, I... (probable)
  3. If I won a million dollars, I... (imaginary)
  4. If you don't eat, you... (fact)
  5. If I see Maria, I... (possible)
  6. If I were you, I... (advice)
  7. Unless you hurry, you... (warning)
  8. If I could fly, I... (impossible)

Section 10: Spot and Correct the Error

ERROR CORRECTION

Say each sentence correctly:

  1. "If it will rain, I'll take an umbrella." → ?
  2. "If I would win the lottery, I would buy a house." → ?
  3. "If I was you, I would go." → ?
  4. "I'll help you if you will ask me." → ?
  5. "If he has time, he will helped you." → ?
  6. "If I would have money, I would travel." → ?
  7. "Unless you don't hurry, you'll be late." → ?
  8. "If you heat ice, it will melting." → ?

Correct answers:

  1. "If it rains, I'll take an umbrella."
  2. "If I won the lottery, I would buy a house."
  3. "If I were you, I would go."
  4. "I'll help you if you ask me."
  5. "If he has time, he will help you."
  6. "If I had money, I would travel."
  7. "Unless you hurry, you'll be late."
  8. "If you heat ice, it melts."
CONSEJO

Recuerda: Zero = If + present, present (hechos). First = If + present, will (posible). Second = If + past, would (hipotético). Nunca uses "will" o "would" en la cláusula "if".
